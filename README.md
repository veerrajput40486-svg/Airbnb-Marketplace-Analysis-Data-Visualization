# Airbnb-Marketplace-Analysis-Data-Visualization

This project focuses on performing a comprehensive End-to-End Data Analysis on an Airbnb dataset. The objective is to clean the raw data, uncover underlying patterns in listing characteristics, explore market pricing dynamics, and track historical user activity trends.

The project leverages Python for the programmatic data cleaning and Exploratory Data Analysis (EDA) phase, and utilizes Tableau to build highly granular, interactive visualizations to answer key business questions.

Tech Stack & Tools Used
Data Manipulation & Cleaning: Python (pandas, numpy)

Exploratory Data Analysis (EDA): Jupyter Notebook

Data Visualization & Dashboarding: Tableau

Core Analytical Questions Answered
The analysis directly addresses the following business questions:

What is the distribution of listing prices? — Identifying price clusters, extreme outliers, and standard market rates.

How are different room types distributed? — Finding out which types of accommodations (e.g., Entire home/apt, Private room, Shared room) dominate the platform supply.

How are listings distributed across different neighborhoods? — Mapping out high-density listing hotspots versus low-supply zones.

What is the relationship between price and room type? — Analyzing how the choice of accommodation directly impacts average pricing tiers.

How has the number of reviews changed over time? — Mapping out historical organic platform growth, tracking seasonal travel waves, and observing major macroeconomic shocks (such as the impact of COVID-19).

Key Steps 
1. Data Extraction
2. Data Cleaning & Descriptive Analysis (Python)
2. Exploratory Data Analysis & Visualizations (Tableau)
